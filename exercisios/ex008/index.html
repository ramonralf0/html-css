Principais Formatações

Negrito / destaque

Nessa frase, temos um termo em negrito usando a tag <b> termo em negrito </b> (não semantica)

Nessa frase, temos um termo em negrito usando a tag <strong> termo em negrito </strong> (semantica)

Italico / Enfase

Nessa frase, temos um termo em italico usando a tag <i> termo em italico </i> (não semantica)

Nessa frase, temos um termo em enfase usando a tag <em> termo em enfase </em> (semantica)

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark> texto marcado </mark>

E no outro parágrafo, temos outro texto marcado.

Texto grande e pegueno

Estamos criando um texto grande e um texto pequeno nesse parágrafo usando a tag <big> texto grande </big> usando a tag <small> texto pegueno </small>

Texto deletado / adicionado

Podemos marcar um texto como escluido para indicar que ele deve ser lido mais não considerado usando a tag <del>texto deletado</del>

Podemos marcar um texto como inserido para dar uma énfase e indicar que ele foi adicionado depois. usando a tag <ins>texto adicionado</ins>

Texto sobrescrito

Para inserir coisas do tipo x20+3, usando a tag <sup>ex: x elevado a 20</sup>

Texto Subscrito

Para inserir coisas do tipo H2O, usando a tag <sub>ex: H2O </sub>
